# Calculadora Estatística

Projeto desenvolvido por **Gabriel** e **Iuri** para a disciplina de **Estatística** na faculdade.

## Sobre o projeto

Esta aplicação tem como objetivo auxiliar no estudo e na prática de conceitos estatísticos por meio de uma calculadora capaz de realizar operações e análises de dados de forma simples e rápida.

## Objetivos

- Facilitar a resolução de cálculos estatísticos;
- Apoiar o aprendizado dos conteúdos da disciplina;
- Oferecer uma ferramenta prática para testes com conjuntos de dados;
- Reforçar a aplicação de conceitos teóricos em um projeto acadêmico.

## Funcionalidades esperadas

Dependendo do andamento do projeto, a calculadora poderá incluir funcionalidades como:

- Cálculo de média;
- Cálculo de mediana;
- Cálculo de moda;
- Variância;
- Desvio padrão;
- Análise básica de conjuntos de dados.

## Público-alvo

Este projeto é voltado principalmente para estudantes e professores que desejam utilizar uma ferramenta simples de apoio em atividades acadêmicas relacionadas à estatística.

## Tecnologias

As tecnologias utilizadas neste projeto podem variar conforme o desenvolvimento, mas a proposta é construir uma solução prática, organizada e fácil de usar.
